# Data Availability Statement

All data and code supporting the findings of this study are openly available as follows:

- The final taxonomically balanced and decontaminated genome collections, phylogenetic marker sets (PMS-Isolate, PMS-HighMAG1, PMS-HighMAG2, and PMS-MediumMAG), supermatrix alignments, and phylogenetic trees are available in this public GitHub repository:  
  https://github.com/[YourUsername]/Asgard_Eukaryote_Data

- Analysis scripts for decontamination, maximum-likelihood inference (IQ-TREE 3), Bayesian inference (PhyloBayes MPI), approximately unbiased (AU) topology tests, and independent post-hoc contamination audit are provided in the `scripts/` directory of this repository.

- This repository has been archived at Zenodo with DOI: [10.5281/zenodo.xxxxxxx] (to be updated upon creating a release).

All data necessary to reproduce the main phylogenetic conclusions and robustness analyses are included in this repository.
